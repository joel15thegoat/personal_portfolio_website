# Responsive Portfolio Website

This repository contains a static portfolio website built with HTML, CSS, and JavaScript.

## Site files
- `index.html` — homepage with portfolio highlights and badge links
- `academic.html` — academic achievements and certifications
- `non-academic.html` — non-academic leadership and community experience
- `competitions_and_enrichment.html` — competitions, enrichment programs, and certifications
- `style.css` — shared site styles
- `script.js` — navigation/menu interactions
- `image.png`, `ids_project_1.png`, `calc_project_2.png`, `keylogger_attempt_project_3.png`, `password_manager_project_4.png` — image assets used by the site

## Notes
- The site is ready for GitHub Pages from the repository root on the `main` branch.
- No duplicate HTML pages were found among the tracked files.
- Non-essential local screenshot files such as `achivement_summary*.jpg` and `Screenshot*.png` are ignored by `.gitignore`.

## How to update
1. Edit the relevant HTML, CSS, or JavaScript file.
2. Use `git status` to review changes.
3. Commit with a clear message and push to `origin/main`.

## GitHub Pages deployment
If the repository is configured for GitHub Pages, the site should publish automatically from the root of the `main` branch.
